What is Lozzalingo?

Lozzalingo is a batteries-included Flask framework that lets you build websites quickly with built-in analytics, authentication, admin dashboard, and more - all working out of the box.

Features

  • Analytics: Automatic page view tracking, device detection, referrer analysis
  • Admin Dashboard: Beautiful admin interface with modular design
  • User Authentication: Email/password login, OAuth (Google, GitHub), password reset
  • News/Blog: Create and publish articles with draft/publish workflow
  • Email Service: Transactional emails via Resend API with templates
  • Customer Spotlight: Instagram-style customer photo gallery
  • Merchandise: Product management system
  • Orders: Order management interface
  • External API: API key authentication for external integrations

Quick Start

from flask import Flask
from lozzalingo import Lozzalingo

app = Flask(__name__)
lozzalingo = Lozzalingo(app)

# That's it! All modules are now available.

Configuration

Customize your site using lozzalingo.yaml: 

site:
  name: "My Site"
  tagline: "Welcome to my site"

features:
  analytics: true
  auth: true
  news: true
  email: true
